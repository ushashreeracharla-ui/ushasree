TaskMate 📋

This is a Flask-based task management application that helps students organize and track their academic tasks efficiently.

Features
User login and registration
Add, update, and delete tasks
Track task completion status
Simple dashboard for task management

Technologies Used
Python (Flask)
HTML, CSS
SQLite Database
JavaScript

How to Run
Install required dependencies
Run app.py
Open localhost:5000 in your browser
